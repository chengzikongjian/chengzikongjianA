# 星光识字岛

一款面向幼儿园及小学阶段的 PWA/H5 识字系统 MVP。首版以孩子端体验为核心，包含学习地图、字卡学习、闯关练习、故事找字、创意工坊、分级字库、专项训练、学情报告、成就系统和本地学习进度。

## 技术栈

- React + TypeScript + Vite
- PWA: vite-plugin-pwa
- 本地持久化: IndexedDB + idb
- 状态管理: Zustand
- 笔顺动画: hanzi-writer
- 图标: lucide-react

## 运行

```bash
npm install
npm run dev
```

## 验证

```bash
npm run lint
npm run build
npm run preview
```

## 当前 MVP 内容

- 52 个样例汉字，覆盖幼儿启蒙到小学五六年级
- 分级课程字库 3500 条：幼儿园 500 字目标 + 小学 1-6 年级 3000 字目标
- 10 个短节奏关卡
- 10 篇分级短故事
- 7 个成就徽章
- 本地学情报告：识字量、星级、课程库掌握率、建议复习字
- 专项训练：轻量跟读测评、书写轨迹练习、标准化字档、分龄游戏矩阵、教材同步和学习计划模板
- 模板化拼句创意工坊

## 后续扩展方向

- 对 3500 条课程字库继续补充人工校对释义、拼音、音频、笔顺和教材单元映射
